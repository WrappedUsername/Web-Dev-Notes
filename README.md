### Find remainder operator %
```JavaScript
/** 
@notice myVariable is assigned a value of 2, because 3 will go into 11, 
3 times with a remainder of 2. 
*/
const myVariable = 11 % 3; 
```
### Increment ++ Decrement --
```JavaScript
myVariable++;
myVariable--;
```
